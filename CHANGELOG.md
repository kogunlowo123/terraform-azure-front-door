# Changelog

All notable changes to this project will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.1.0/),
and this project adheres to [Semantic Versioning](https://semver.org/spec/v2.0.0.html).

## [1.0.0] - 2024-01-15

### Added

- Azure Front Door profile with Standard and Premium SKU support
- Endpoint management with enable/disable controls
- Origin groups with health probes and load balancing configuration
- Origins with priority, weight, and private link support
- Route configuration with forwarding protocol and HTTPS redirect
- Custom domain support with managed TLS certificates
- Rule sets with conditions and actions (URL redirect, rewrite, headers)
- Caching with query string behavior and compression
- WAF policies with custom and managed rules
- Security policy association for WAF-to-endpoint binding
- Rate limiting rules
- Custom domain association and validation
- Basic, advanced, and complete usage examples
